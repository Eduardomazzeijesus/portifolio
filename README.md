# Portfólio Profissional — Desenvolvedor em Formação

Um portfólio moderno e responsivo desenvolvido com **HTML, CSS e JavaScript puro** (sem frameworks).

## 🎨 Design

- **Estilo**: Neomorfismo Escuro com Acento Ciano
- **Paleta**: #1A1A2E (fundo), #16213E (superfície), #00F5D4 (ciano), #7B2FBE (violeta)
- **Tipografia**: Syne (display) + DM Sans (body) + Fira Code (mono)
- **Responsivo**: Totalmente adaptável para mobile, tablet e desktop

## 📁 Estrutura

```
portfolio-estruturado/
├── html/
│   └── index.html       # Estrutura HTML
├── css/
│   └── styles.css       # Estilos CSS
├── js/
│   └── script.js        # Lógica JavaScript

```

## 🚀 Como Usar

1. **Abrir no navegador**: Basta abrir o arquivo `html/index.html` em qualquer navegador moderno
2. **Personalizar**: Edite o arquivo `html/index.html` para adicionar seus dados:
   - Nome
   - E-mail
   - GitHub/LinkedIn
   - Descrição pessoal
   - Projetos
   - Formação

3. **Customizar cores**: Edite as variáveis CSS em `css/styles.css`:
   ```css
   :root {
       --primary-bg: #1A1A2E;
       --cyan: #00F5D4;
       --violet: #7B2FBE;
       /* ... */
   }
   ```

## 📝 Seções

### Hero
- Apresentação com efeito typewriter
- Partículas animadas em canvas
- Avatar com glow animado
- Estatísticas rápidas

### Sobre
- Descrição pessoal
- Interesses (tags)

### Habilidades
- 8 tecnologias com barras de progresso
- Ícones de ferramentas
- Animações ao scroll

### Projetos
- 6 projetos em cards neomórficos
- Descrição, tags e emoji
- Hover effects

### Formação
- Timeline vertical
- 4 marcos da jornada
- Design responsivo

### Contato
- Cards com informações de contato
- Formulário funcional
- Validação de campos

## 🎯 Personalizações Recomendadas

1. **Adicione seus dados reais** no HTML
2. **Links dos projetos** — adicione URLs reais
3. **Foto de perfil** — substitua o avatar
4. **Cores** — adapte a paleta para sua marca pessoal
5. **Certificados** — crie uma nova seção para cursos online
6. **Blog/Artigos** — adicione uma seção de posts
